# Architecture

SILENCE.OBJECTS is a pnpm monorepo with:
- `apps/` — Next.js applications (portal, patternlens, patternslab)
- `packages/` — Shared libraries (@silence/* namespace)
- `agents/` — Autonomous agent army
- `tooling/` — Shared configs (TypeScript, ESLint, generators)

See CLAUDE.md for current implementation status and gap analysis.
