# SILENCE.OBJECTS — 12 ARCHETYPES SYSTEM

System klasyfikacji wzorców behawioralnych (NIE osobowości) oparty o 12 archetypów Jungowskich.

## Lista

Creator, Ruler, Caregiver, Explorer, Sage, Hero, Rebel, Magician, Lover, Jester, Innocent, Orphan.

Każdy: dominant pattern, typical tension, shadow pattern, growth edge.

## Scoring i blends

- mapArchetype(): scoring 0–100 per archetype.
- scoreBlend(): opisowe blendy (np. Creator/Hero).
- trackShift(): timeline przesunięć w czasie.

## Framing rules

- ZAWSZE: "Your patterns currently align with Creator/Hero blend..."
- NIGDY: "You are a Creator"
- Pojęcia: Pattern, Tension, Function — NIE "traits", "personality"
- Mapping = strukturalna narracja, NIE diagnoza.
